# Alpha Trade

A deep reinforcement learning trading bot and backtesting framework. We train a custom PPO (Proximal Policy Optimization) agent from scratch to trade a portfolio of 20 S&P 500 stocks, then deploy it to paper trade via Alpaca.

## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-org/Alpha_Trade_MDST.git
cd Alpha_Trade_MDST

# Install dependencies
pip install -r requirements.txt

# Download stock data
jupyter notebook load_data.ipynb   # Run all cells

# Train the agent
jupyter notebook train.ipynb       # Run all cells
```

## Technical Details

**Environment**: Custom Gymnasium env simulating a 20-stock portfolio with transaction costs. Actions are continuous values in [-1, 1] per stock (sell to buy).

**Agent**: Actor-Critic network with tanh-squashed Gaussian policy.

**Training**: PPO with Generalized Advantage Estimation (GAE), clipped surrogate objective, entropy bonus, and gradient clipping.

**Indicators**: RSI, Bollinger Bands, EMA-12/26, MACD, Relative Volume.
